unsigned int *  rand_nums 
)

Hash function for Slater determinants in bit-string representation.

Calculates the Merkle-Damgard hash value from occupied orbitals in determinant, according to Algorithm 1 in Booth et al. (2014)

Parameters
[in]occ_orbslist of occupied orbitals in the determinant (length n_elec)
[in]n_elecNumber of occupied orbitals in the determinant
[in]rand_numsList of random integers used in hash function, must have at least (max(occ_orbs) + 1) elements
Returns
the calculated hash value
