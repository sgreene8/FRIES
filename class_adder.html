Detailed Description

template<class el_type>
class Adder< el_type >

Class for adding elements to a DistVec object.

Template Parameters
el_typeType of elements to be added to the DistVec object Elements are first added to a buffer, and then the buffered elements can be distributed to the appropriate process by calling perform_add()
